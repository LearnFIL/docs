# Roadmap and Project Status

## What is complete

- Browser based learning app
- Local execution and validation engine
- localStorage based progress tracking
- Modular lesson structure
- Basic UI and navigation

## What is in progress

- Telegram bot integration
- Discord bot integration
- Filecoin based checkpoint storage
- libp2p event propagation
- Learning milestone broadcasting
- Curriculum snapshot design

## What is coming next

### Short term
- Trusted emitter configuration
- End to end publish and receive flow
- Stable bot commands
- CID based progress sharing
- libp2p peer discovery

### Medium term
- Verifiable completion certificates
- Curriculum versioning
- Cross device progress sync
- Agent driven learning suggestions

### Long term
- Community hosted indexers
- Multi community learning networks
- Portable learning credentials
- Advanced Filecoin and storage modules

## Guiding principle

LearnFIL evolves by:
- Adding decentralization
- Avoiding centralization
- Keeping the learner in control
- Making learning verifiable and portable
